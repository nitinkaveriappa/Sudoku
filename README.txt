Compilation Steps:

In the terminal do

g++ sudoku.cpp -o sudoku

Run Command :

./sudoku a1/a2/a3/a4/a5 in.txt out.txt

where,

a1=Backtrack algorithm

a2=Backtrack with forward checking

a3= Arc consistency 3

a4=Vertex Order heuristic

a5=Simulated Annealing



in.txt is the input file to be given having the initial configuration of the board.
out.txt is the output file where the output - solved sudoku board is requird to be written.
Both inputfile and the empty output file need to be created previously before running the run command.
